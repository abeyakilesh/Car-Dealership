# Veloce Motors - Premium Automotive Website

**A high-performance, responsive frontend website designed for a luxury automobile brand.**

This project demonstrates a premium UI/UX design, featuring a custom 3D javascript slider, dynamic content rendering, and client-side form validation, built entirely HTML,BootStrap CSS, Vanilla JavaScript.

## Key Features

* **Premium UI Design:** Minimalist, "Porsche-style" aesthetic using Bootstrap 5 and custom CSS.
* **3D Interactive Slider:** Custom Vanilla JS carousel with center-focus scaling and smooth transitions.
* **Dynamic Details Page:** A single `model-details.html` page that renders different content based on URL parameters.
* **Mock Authentication:** Functional Login/Logout system using `localStorage` to persist user sessions.
* **Contact Form:** Client-side validation with success/error states.
* **Responsive:** Fully mobile-optimized layout using Bootstrap's grid system.

## Technologies Used

* **HTML5** - Semantic markup.
* **CSS3** - Custom animations, gradients, and layout styling.
* **Bootstrap 5** - Responsive grid and components (Navbar, Modals).
* **Vanilla JavaScript** - DOM manipulation, URL parsing, and logic (No jQuery).

## Project Structure
